첫 문단은 도입부입니다. 여기에는 제목(##)을 쓰지 말고, 글이 무엇을 다루는지
2~3문장으로 편하게 풀어 쓰면 카드 요약과 잘 어울립니다.

## 이렇게 h2 로 큰 단락을 나눕니다

`## 제목` 과 `### 제목` 은 **오른쪽 목차에 자동으로** 들어갑니다.
(h1 은 글 제목이 이미 쓰므로 본문에서는 h2 부터 시작하세요.)

강조는 **굵게**, *기울임*, ~~취소선~~, 그리고 `인라인 코드` 를 쓸 수 있습니다.

### 이렇게 h3 로 하위 단락을

문단과 문단 사이는 **빈 줄 하나**로 나눕니다. 빈 줄이 없으면 한 문단으로 붙습니다.

## 목록

순서 없는 목록:

- 첫 번째 항목
- 두 번째 항목
  - 들여쓰기(스페이스 2칸)로 하위 항목
- 세 번째 항목

순서 있는 목록:

1. 준비하기
2. 실행하기
3. 확인하기

체크리스트(할 일 목록):

- [x] 끝난 일
- [ ] 아직 안 한 일

## 코드 블록

백틱 3개 뒤에 **언어 이름**을 적으면 문법 강조가 자동으로 됩니다.

```ts
// 언어를 적으면(ts, js, py, sql, go, bash, json …) 색이 입혀집니다
function greet(name: string): string {
  return `안녕하세요, ${name}님`;
}
```

## 인용

> 인용문은 문장 앞에 > 를 붙입니다.
> 팁이나 중요한 문장을 강조할 때 좋습니다.
